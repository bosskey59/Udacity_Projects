Description

Diamond Weapons Safe is a prototype for a technologically advanced, underground safe for firearms. It communicates via bluetooth to an android application which will allow the user to unlock and lock firearms with the correct numeric code. The safe will also provide the user with alerts when the temperature or humidity is outside a given threshold.
