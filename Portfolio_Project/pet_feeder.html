Network Cat Feeder

Tools/Frameworks used

Raspberry Pi, Servo , Cereal Dispenser, usb wifi adapter

Description

This will be an on going project and I will continue to update this page while I progress through the project. My goal is to make a network cat feeder capable to dispensing dry food through a web app. I will use a raspberry pi to control a servo which will be doing the dispensing. The raspberry pi will connect to my local home network via wifi and I will build a web application to send commands from my mobile device.

9/8/16

Purchased most materials needed for the project. Recieved cereal dispenser, servo, raspberry pi, and wifi module, sd card, hdmi cord. I will proceed to do the initial setup of the raspberry pi and start researching how to send the correct signals to the servo to get it working.
