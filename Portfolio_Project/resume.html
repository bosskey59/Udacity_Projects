Online Resume

Tools/Frameworks used

jQuery, JSON, Javacript HTML, CSS

Description

The purpose of this project was to learn how to use jquery to and json. I stored all of the data you see on this page in JSON and then use jquery to iterate through it and display it where it needs to be. This allows me to quickly update my resume without modifying the javascript functions that iterates through the data. Instead you can simply add data to the respective JSON objects and the same javascript will display it properly onto the page.
